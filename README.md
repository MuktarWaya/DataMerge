# DataMerge

Static browser app for merging HDC `exchange_file` data with Mypcu `rptPersonTypeName` data.

## Cloudflare Pages settings

- Framework preset: `None`
- Build command: leave blank
- Build output directory: `/`
- Root directory: `/`
- Production branch: `main`

The app runs fully in the browser for file processing. Google Apps Script files are kept only as the previous backend/log reference.
